# Raw Dataset

## Dataset Information

- Dataset Name: BDD User Story Dataset (500 User Stories)
- Source: Dataset constructed from the study by Rath et al.
- Original File: Requirements.xlsx
- License: Academic research and educational use.

## Description

This folder contains the original dataset of 500 user stories used in this study.
The dataset was collected from the work of Rath et al. and stored in `Requirements.xlsx`.

The original dataset is kept unchanged to ensure reproducibility. All preprocessing, sampling, and annotation are performed on copies of the dataset.

## Dataset Structure

The dataset contains the following columns:

- ID
- User Story
- Requirements
- Manual Scenario

## Derived Files

The following files are generated from the original dataset:

- pilot_sample.csv
- pilot_ground_truth.csv
- full_ground_truth.csv

## Notes

- The original dataset must not be modified.
- A random sample of 100 user stories is used for the main experiment.
- A pilot sample of 20 user stories is used for pilot evaluation.
